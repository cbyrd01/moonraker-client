# Changelog

## 0.1.0 (2026-04-08)

Initial release.

- Complete REST API coverage for Moonraker (133+ endpoints, 12 API categories)
- Synchronous and asynchronous client classes
- WebSocket support with JSON-RPC 2.0 and notification subscriptions
- 30 dataclass models generated from the OpenAPI spec
- 15 high-level helper functions for common printer operations
- API key, JWT, and oneshot token authentication
- Full type annotations with PEP 561 py.typed marker
- 155 tests (122 unit + 33 functional)
